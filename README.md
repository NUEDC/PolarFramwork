# PolarFramwork

Polar构架代码库
Polar构架是一个基于MCU的集成底层代码库

其驱动层适用于以下设备：
STM32F1ZET6
STM32F4

其算法层适用于CMSIS标准的设备,使用FIR等滤波器的时候应当使用带有FPU和DSP的处理器
